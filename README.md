AI collaboration platform


## Developing

### Requirements
1. Docker installed

### Start environment

`git clone git@github.com:dark-side/ai-collaboration-platform.git`

`cd ai-collaboration-platform`

`make build`

`make start`


### Run rails console
`make console`

### Run rubocop
`make lint`

### Connect to bash in container
`make bash`

### Run specs
`make bash`

`bundle exec rspec ...`
